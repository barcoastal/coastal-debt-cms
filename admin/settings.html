Setup Instructions

  1. Go to Google Ads API Center and apply for a Developer Token (Basic access is fine)
  2. Go to Google Cloud Console and create OAuth 2.0 credentials
  3. Set the redirect URI to: http://localhost:3000/api/google-ads/callback
  4. Enable the Google Ads API in your Google Cloud project
  5. Set environment variables: GOOGLE_ADS_CLIENT_ID and GOOGLE_ADS_CLIENT_SECRET
  6. Enter your Developer Token above and click Save
  7. Click "Connect Google Ads Account" to authorize
